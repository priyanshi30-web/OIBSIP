QUIZ APP

📱 Overview
Quiz App is an Android application developed using Java in Android Studio. It allows users to test their knowledge by answering multiple-choice questions from different categories. At the end of the quiz, the app displays the user's score and percentage.

✨ Features
Welcome screen
5 quiz categories
Multiple-choice questions (4 options)
Correct answer highlighted in green
Wrong selected answer highlighted in red
1.5-second delay before the next question
Progress bar
Final score and percentage
High score saved using SharedPreferences
Questions loaded from JSON files

🛠️ Technologies Used
Java
Android Studio
XML
Gson Library
JSON
SharedPreferences

📂 Categories
General Knowledge
Science
Geography
Computer
Sports

🚀 How to Run
Open the project in Android Studio.
Sync Gradle.
Run the application on an emulator or Android device.
Select a category and start the quiz.
