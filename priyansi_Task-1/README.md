📱 Unit Converter App

📌 Overview

The Unit Converter App is an Android application built using Java and XML in Android Studio. It allows users to convert values between different units with a clean, simple, and user-friendly interface.

✨ Features

- 📏 Length Conversion
- ⚖️ Weight Conversion
- 🌡️ Temperature Conversion
- ⏱️ Time Conversion
- 🚗 Speed Conversion
- Dynamic unit selection
- Material Design UI
- Fast and accurate conversions
- HashMap-based conversion logic

🛠️ Technologies Used

- Java
- XML
- Android Studio
- Material Design
- CardView
- HashMap

🚀 How to Use

1. Select a category.
2. Choose the From and To units.
3. Enter the value.
4. Tap Convert.
5. View the converted result.

📈 Complexity

- Time Complexity: O(1)
- Space Complexity: O(1)
