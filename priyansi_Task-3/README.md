Calculator App

Overview

A simple Android Calculator application developed using Java and Android Studio. It performs basic arithmetic operations with a clean and user-friendly interface.

Features

- Addition, Subtraction, Multiplication, and Division
- Decimal number support
- Input validation
- Division by zero handling
- Clear (CE) functionality
- Simple and responsive UI

Technologies Used

- Java
- Android Studio
- XML

How to Run

1. Clone or download the project.
2. Open it in Android Studio.
3. Build and run the application on an emulator or Android device.

Learning Outcomes

This project helped me understand Android UI design, event handling, Java programming, resource management, and basic calculator logic.
